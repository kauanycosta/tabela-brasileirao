# 🏆 Tabela do Brasileirão 2025

Projeto de uma **tabela interativa do Campeonato Brasileiro de Futebol 2025**, desenvolvido com **PHP**, **HTML**, **CSS** e **JavaScript**. Os dados são lidos a partir de uma **API**, permitindo exibição atualizada dos clubes, resultados recentes e informações de próximos jogos.

## 🌟 Funcionalidades

- Consumo de API via **PHP** para montar a tabela com dados reais
- Visualização interativa da tabela: posição, time, pontos e desempenho
- Exibição de **cards informativos** com:
  - Últimos 5 resultados do time
  - Informações sobre o próximo jogo
- Suporte total a **dispositivos móveis e desktops**
- Interações via **hover (desktop)** e **clique (mobile)**
